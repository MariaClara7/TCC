Estrutura

Essa é a estrutura inicial de um projeto HTML.

                <!DOCTYPE html>
                <html>
                <head>
                <title>Page Title</title>
                </head>
                <body>
                
                <h1>My First Heading</h1>
                <p>My first paragraph.</p>
                
                </body>
                </html>
pink green blue orange

No entanto, para colocar esse pedaço de código HTML dentro do nosso projeto, foi necessário encapsular o código na tag <pre>, e utilizar os marcadores de texto &lt; e &gt; que são interpretados como < e >.

Sem essas entidades, o HTML entenderia como um código funcional.
